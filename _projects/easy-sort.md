## Overview

A stable, actively maintained mod (currently v1.4.1) supporting multiple
Minecraft version branches, built with configurable sort ordering as a core
design goal.

## Highlights

- One-hotkey sorting, quick-stack, and restock
- Published on Modrinth and CurseForge, with jars on GitHub Releases
