## Overview

Built during a software engineering internship. Integrates with DigiSoil
(texture) and DigiFert (NPK) hardware sensors plus real-time weather and news
feeds, backed by Firebase (Auth, Firestore, Storage, Cloud Functions) and a
Cloud Run service for photo-based soil analysis.
